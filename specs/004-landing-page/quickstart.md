# Quickstart: Landing Page

## How to use

1. **Visit Root URL**: Navigate to `http://localhost:3000/`.
2. **If Logged Out**: You should see the new landing page with Hero, Features, and Footer sections.
3. **If Logged In**: You should be automatically redirected to `/dashboard`.
4. **CTA Action**: Click "시작하기" to go to the login page.

## Development Verification

- **Unauth Test**: Clear browser cookies or use Incognito mode to view the landing page.
- **Auth Test**: Log in, then try to navigate back to `/`. You should be redirected to `/dashboard`.
- **Responsive Test**: Check the layout on mobile screen widths (375px - 414px).
