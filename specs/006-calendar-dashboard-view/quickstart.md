# Quickstart: Calendar Dashboard View

## Verification Steps

### 1. View Toggle
- Navigate to `/dashboard`.
- Find the toggle icon (Calendar/List) near the search bar.
- Click to switch between List and Calendar views.

### 2. Calendar Display
- Verify the calendar grid is rendered correctly for the current month.
- Use the arrow buttons to navigate to previous/next months.

### 3. Entry Markers
- Ensure days with existing entries have a hand-drawn circle marker.
- Hover/Click the marker to ensure it leads to the correct entry detail page.

### 4. Responsiveness
- Test on mobile view to ensure the calendar grid scales correctly without horizontal scrolling.
